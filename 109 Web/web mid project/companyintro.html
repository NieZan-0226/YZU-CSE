服務理念 :

看見從花蓮上來的阿美族林學榮同學

只帶了幾件衣服而已

最近天氣偏冷了想幫他買個衣服

而經費全來自各位在咱們店家的消費!!!!

希望大家一起跟著我們 救學榮!!!!!

產品製作:

這些原住民傳統服飾是由各族的族人所製作的

而要怎麼運過來賣呢?

各位不用擔心我們不雇用童工或外勞

我們雇用原住民的驕傲---林學榮!!!

員工福利:

1.帶原住民闖蕩台灣

2.會扣員工的薪水，買他們在平地要穿的衣服送他們
